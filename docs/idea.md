- **일반**
- **빠른** -> 빨리 죽이기 or 둔화
- **돼지** -> 쌘거 때리기
- 은신 (+평타무시?) -> 광역기 or 은신 해제 기믹
- 방패병 (전면 공격 방어? 일정 범위 보호막?) -> 방패를 때려서 까기
- 도발 (평타만) -> 오기 전에 죽이기 or 평타 강화
- 마법무시 -> 평타로 죽이기
- 평타무시 -> 오기 전에 죽이기
- 분노 (맞으면 빨라짐?) -> 한번에 죽이거나, 잘 때리기
- 마법 봉쇄 (시전 시작 후 일정 시간 지나면 플레이어 마법 일정 시간 잠금) -> 봉쇄하기 전에 죽이기
- 보호막만 부수는 자폭병? -> 오기 전에 죽이기
- 약화 필요한 적 (디버프 없으면 단단, 있으면 물렁해짐)
- 소환사 -> 단일기로 빨리 죽이든, 광역기로 말리든
- 시점 고정 (화면 못 바꿈) -> 하기 전에 죽이기
- 투척병
